Introduction

{{ app_name }} is a comprehensive solution for creating professional QR codes for business cards and visiting cards. With our platform, you can:

  • Generate unlimited QR codes
  • Store contact information in vCard format
  • Customize colors, sizes, and formats
  • Save and reuse templates
  • Track downloads and views

Getting Started

1. Create an Account

Register for a free account to start generating QR codes.

2. Login

Access your dashboard using your credentials.

3. Navigate

Use the sidebar menu to access different features.

Generating QR Codes

To generate a new QR code:

  1. Click Generate QR Code in the sidebar
  2. Enter a name for your QR code
  3. Fill in the contact information fields
  4. Choose customization options (optional)
  5. Click Generate QR Code
Contact Information Fields
  • Name: Full name of the contact
  • Title: Job title or position
  • Company: Company or organization name
  • Email: Email address
  • Phone: Phone number
  • Website: URL of website
  • Address: Physical address

Managing QR Codes

Access My QR Codes to view and manage all your generated QR codes.

Available Actions
  • View: See detailed information about a QR code
  • Edit: Update name, description, and category
  • Download: Download the QR code file
  • Delete: Remove a QR code permanently
Search and Filter

Use the search bar to find specific QR codes by name or description. Filter by category to organize your codes.

Templates

Templates allow you to save your favorite QR code settings for quick reuse.

Creating a Template
  1. Navigate to Templates
  2. Click Create Template
  3. Set your preferred colors and settings
  4. Save the template
Using Templates

When generating a new QR code, select a template from the dropdown to apply its settings automatically.

Customization Options

Colors
  • Foreground Color: The color of the QR code pattern
  • Background Color: The background color

Ensure good contrast for optimal scanning.

Size

Set the dimensions of the QR code in pixels. Default is 300x300px.

Error Correction
  • Low (7%): Smallest size, minimal damage tolerance
  • Medium (15%): Balanced option
  • Quartile (25%): Good for print
  • High (30%): Best for logos, highest damage tolerance
File Format
  • PNG: Raster format, best for web
  • SVG: Vector format, scalable
  • PDF: Document format, ideal for printing

Best Practices

Design
  • Use high contrast colors (black on white is ideal)
  • Ensure adequate size for the intended use
  • Test your QR code before printing
  • Leave white space around the QR code
Printing
  • Use SVG or PDF for best print quality
  • Choose High error correction for printed codes
  • Minimum size for printing: 2cm x 2cm
  • Test scan from the expected viewing distance
Content
  • Keep contact information up to date
  • Include essential details only
  • Use standard formatting for phone numbers and emails
